# pacemanbot

本插件基于 Astrbot 开发

## 功能

1. /register 用户名-将玩家 IGN 和 QQ 号绑定
2. /paceman [用户名]-查询某玩家的 24 小时数据（不添加用户名则查询自己）
3. /rank [用户名]-查询某玩家 rank 数据（不添加用户名则查询自己）
4. 定时设定榜单（请联系管理员开通）
