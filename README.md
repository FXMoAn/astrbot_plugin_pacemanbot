# pacemanbot
## 项目介绍
本项目为基于AstrBot开发的插件，用于查询我的世界JAVA版速通信息（以CN为主）
作者：墨安

## 使用方法
基本命令有:
/bothelp-查询指令帮助
/paceman 用户名-获取该用户24小时内的paceman数据（暂时包括下界、盲传和结束）
/ldb 参数
    -nether 获取下界数量排行
    -finishtime 获取完成平均时间排行
    -finishavg 获取完成数量排行
/rank 用户名-获取rank信息
